Hamiltonian Cycle in a Graph

A cycle that visits each vertex exactly once and returns to the starting vertex.

  • Useful for route, outline, and structural loop design.
  • Great for optimized tour-style paths.

Eulerian Circuit in a Graph

A closed path that visits every edge exactly once and returns to the starting vertex.

  • Useful for robot drawing without lifting the tool.
  • All vertices must have even degree.
